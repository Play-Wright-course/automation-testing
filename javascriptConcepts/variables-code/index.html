    JavaScript: var, let, const – Code Snippets with Output¶

    1. var Scope¶

    function testVarScope() {
  if (true) {
    var x = 10;
  }
  console.log("var x =", x);
}
testVarScope();

    Output:

    var x = 10

    2. let Scope¶

    function testLetScope() {
  if (true) {
    let y = 20;
    console.log("Inside block y =", y);
  }
}
testLetScope();

    Output:

    Inside block y = 20
